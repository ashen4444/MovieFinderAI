MovieFinderAI is an AI-powered movie discovery and recommendation system designed to go beyond traditional keyword-based search.

The system leverages modern techniques such as semantic embeddings and hybrid search (keyword + vector search) to understand user intent and provide highly relevant movie recommendations.

🎯 Key Features

🎬 Recommend movies based on titles and descriptions

🧠 Semantic understanding using embedding models

🔍 Hybrid search with Elasticsearch + vector similarity

📊 Uses large-scale real-world dataset (TMDB + IMDb)

📅 Focus on modern movies (post-2000)

⚙️ Tech Stack (Planned)

Python

Pandas (data preprocessing)

OpenAI Embeddings (text-embedding-3-small)

Elasticsearch (vector + keyword search)

Streamlit (UI - optional)

📊 Dataset

The Ultimate 1Million Movies Dataset (TMDB + IMDb)

🚀 Goal

To build a production-like intelligent recommendation system similar to what streaming platforms use, while learning real-world AI system design.
